# Java JDBC Employee Database App

## Objective
This project demonstrates CRUD operations using Java JDBC.

## Technologies Used
- Java
- MySQL
- JDBC

## Features
- Add Employee
- View Employees
- Update Employee
- Delete Employee

## How to Run
1. Create database and table using SQL script.
2. Update DB credentials in DBConnection.java.
3. Compile and run MainApp.java.

## Outcome
Learned how to connect Java with database and perform CRUD operations using JDBC.
